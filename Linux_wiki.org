* File System
** Disk and Directory capacity
   1. df : report file sys disk space usage
      -h, human read
   2. du : list all file's capacity (unit: 1K)
** Hard link and Symbolick link
*** Hard link :
Equal to copy the ordinary file, same inode to original file.
| 34474855 -rw-r--r--. 2 root root 451 Jun 10  2014 crontab      |
| 34474855 -rw-r--r--. 2 root root 451 Jun 10  2014 /etc/crontab |                                                                |
**** limit on Hard link:
1. Can't use on diff.
2. Can't link a directory.
*** Symbolic Link :
Build a new file(new inode), it would point to orignal file.
| 34474855 -rw-r--r--. 2 root root 451 Jun 10  2014 /etc/crontab                   |
| 53745909 lrwxrwxrwx. 1 root root  12 Jun 23 22:31 /root/crontab2 -> /etc/crontab |
**** Flaw on SL :
1. It would be malfuntion if original is deleted.

** Login or Non-login shell
*** Login
    tt1~tt6, account and password are required.
**** /etc/profile
     Default system setting
     it would also called the other files:
***** /etc/profile.d/*.sh
***** /etc/locale.conf
      Default language setting file
***** /usr/share/bash-completion/completions/*
      [tab]shortcut, auto-command fill up
**** ~/.bash_profile
     System would read each files below, if one
     of them exist, it would stop reading:
     1. ~/.bash_profile
     2. ~/.bash_login
     3. ~/.profile
**** source
     When you add some personal setting on .bash_profile,
     but don't want to relogin.
     | source ~/.bashrc |
     | .  ~/.bashrc     |
*** Non-login
    Enter an environment without account and password.
    System would only read =~/.bashrc=
** File redirect
*** Standard output(STDOUT) & Standard error output(STDERR)
    Copy normal or error screen message and extract to a file.
    | Pattern    | Code | Command                            |
    |------------+------+------------------------------------|
    | stdin      |    0 | < or <<                            |
    | stdout     |    1 | > or >>                            |
    | stderr     |    2 | 2> or 2>>                          |
    |------------+------+------------------------------------|
    | overwrite  |      | >                                  |
    | add        |      | >>                                 |
    | 1&2        |      | (filename) > (filename) 2>&1 or &> |
    | black hole |      | /dev/null                          |
*** Standard input(STDIN)
    Import one file's content to another file
    Ex :
    =cat > catfile < ~/.bashrc=  // Overwrite catfile by .bashrc
    =<< "char"= // Overwrite until "char"
